# 🎙️ Diponegoro Recorder

**Diponegoro Recorder** adalah alat bantu berbasis web yang dirancang untuk mempermudah pengerjaan tugas speaking pada platform **Learnsocial**. Website ini memungkinkan pengguna untuk merekam suara, memantau volume, dan mengunduh hasilnya langsung dalam format MP3.

## Fitur Utama
- **Perekam Suara:** Rekam suara berkualitas tinggi langsung dari browser.
- **Indikator Volume:** Visualisasi level suara secara real-time.
- **Timer Otomatis:** Menghitung durasi rekaman dengan presisi.
- **Format MP3:** Hasil rekaman otomatis dikonversi ke format `.mp3`.
- **Desain Responsif:** Tampilan yang optimal baik di komputer maupun perangkat mobile (HP).

## Cara Penggunaan
1. Buka file `index.html` di browser Anda.
2. Klik tombol **"Mulai Rekam"** dan berikan izin akses mikrofon jika diminta.
3. Perhatikan indikator volume dan timer saat Anda berbicara.
4. Klik **"Berhenti"** setelah selesai.
5. Dengarkan kembali hasil rekaman pada pemutar audio yang muncul.
6. Klik **"Unduh File MP3"** untuk menyimpan hasil rekaman Anda.

## Teknologi yang Digunakan
- **HTML5 & CSS3:** Struktur dan desain antarmuka.
- **JavaScript (Vanilla):** Logika perekaman dan manipulasi Audio API.
- **Lamejs:** Library untuk encoding MP3 di sisi klien.
